# Subscription_churn_analysis
End-to-end data analysis project using Python, SQL, and Power BI to identify revenue leakage and predict customer churn in a subscription-based business.
**Project Goal:**
This project analyzes a dataset of 500 customers to quantify 'Revenue Leakage' money lost when customers cancel subscriptions and identifies 'At-Risk' users before they leave.
**Tech Stack used:**
Python: Data generation (Faker) and Correlation Analysis (Seaborn/Matplotlib).
MySQL: Database schema design and complex analytical querying.
Power BI: Interactive dashboarding and DAX-based KPI calculation.
